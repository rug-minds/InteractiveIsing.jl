# MNIST Aggregate Plots

Use of this folder: aggregate comparison plots for this architecture family. Per-experiment plots live beside the CSV files under `experiments/current/<experiment-name>`.

Checkpoint and post-hoc evaluation CSVs are plotted only in their experiment folders because reads/sweeps differ between runs.

Line styles encode NN/radius groups: solid for 1-3, dashed for 4-7, dotted for 8 and above. Colors are high-contrast and reused only when many configurations are shown.
