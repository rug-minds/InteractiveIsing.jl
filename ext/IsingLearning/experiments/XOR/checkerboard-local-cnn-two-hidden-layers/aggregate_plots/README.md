# Checkerboard Aggregate Plots

This folder contains comparison PNGs collected from retained checkerboard XOR
runs. The source data lives in the corresponding run folders under
`experiments/current`.

Line styles encode local connection radius:

- radii 1 through 3: solid, thinner lines
- radii 4 through 7: dashed, medium lines
- radii 8 and above: dotted, thicker lines

These plots are visual summaries only. Use each run folder's `metrics.csv` and
`summary.csv` for the underlying values.
