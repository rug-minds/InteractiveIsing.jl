# Diagnostics Individual Plots

Output-code folders use explanatory names:
- `output_majority_vote`: scalar/majority-style output readout
- `output_spatial_pattern_target`: output layer trained toward a spatial target pattern
- `output_replicated_two_class`: replicated false/true output code
